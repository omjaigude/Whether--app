## 🌦️ Weather App

A simple weather app made by a om jaigude.  
It shows the temperature, humidity, and weather condition of any city.

## 🛠️ Built With
- HTML  
- CSS  
- JavaScript  

## 💡 How to Use
1. Open the app in your browser.  
2. Type a city name.  
3. Click Search to see the weather.
